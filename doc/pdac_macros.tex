\newcommand{\UGDESC} {
The \PDAC\ {\em\UGTITLE\/} describes how to run and use the 
various features of the Piroclastic Density Analysis Code \PDAC.  
This guide includes the capabilities of the program, how 
to use these capabilities, the necessary input files and 
formats, and how to run the program both on uniprocessor 
machines and in parallel.}

\newcommand{\RMDESC} {
The \PDAC\ {\em\RMTITLE\/} is intended for the user who wants
to modify the features of \PDAC or to the programmers who want to 
cooperate to the \PDAC development. It describes how the model equations
are discretized and solved, it provides the explicative list of variables 
used in the numerical code and gives a short description of the 
parallelization strategy, so that the advanced user can optimize the
code efficiency on some computational platform.}
